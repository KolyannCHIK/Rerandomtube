# Инструкция по сборке приложения для Windows

Этот проект теперь поддерживает сборку в виде автономного приложения для Windows (exe) с использованием Electron.

## Предварительные требования

1.  Установите [Node.js](https://nodejs.org/) (рекомендуется версия LTS).
2.  Установите зависимости проекта, выполнив команду в терминале:
    ```bash
    npm install
    ```

## Запуск в режиме разработки

Чтобы запустить приложение в режиме разработки (с горячей перезагрузкой):

```bash
npm run electron:serve
```

## Сборка для Windows

Чтобы собрать готовый `.exe` файл (установщик и портативную версию):

```bash
npm run electron:build
```

После завершения сборки файлы будут находиться в папке `dist_electron`.

### Возможные проблемы при сборке

-   **Ошибка скачивания бинарников Electron**: Если во время установки зависимостей или сборки возникают ошибки сети, попробуйте использовать VPN или зеркала.
-   **Иконка**: Убедитесь, что иконка приложения `public/splash_screens/icon.png` существует. Она используется для сборки.

## Конфигурация

Настройки сборки Electron находятся в файле `vue.config.js` в секции `pluginOptions.electronBuilder`. Там можно изменить:
-   `productName`: Название приложения.
-   `appId`: Уникальный идентификатор приложения.
-   `win.icon`: Путь к иконке.
